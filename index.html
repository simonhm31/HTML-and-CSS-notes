Measurement Units:

Absolute Measurement Units-

Pixels


Relative to Viewport Size-

vh, vw, %


Relative to Font Size-

em, rem


VH is viewport height VW is viewport width. These work differently than % because % is relative to the parent element whereas vh and vw are relative to the viewport

1 em is equal to the current font size, whatever that may be. 1 rem is equal to the root font size, which is always 16 pixels.

Positioning:

The default value for the position CSS keyword is "static".

These boxes all have position: static

These boxes all have position: relative

With position: relative we can position elements relative to their original (or static) position

This is done with the left, right, top, and bottom keywords

These boxes all have position: absolute

This makes child elements invisible to their parent container.
This positions elements relative to their container ONLY IF the parent container has the position: relative property

If the container has position:static the child elements won't appear at all

This is done with the left, right, top, and bottom keywords

These boxes all have position: fixed

This positions elements relative to the viewport window

For example, the gold box is way up in the top left corner of this page, and the red and blue boxes are in the top right.

If you want something to stay on the viewport screen no matter what, this is how to do it.

Linking HTML files together:

HTML files can be linked together with the "href" attribute

ONLY these tags can support the href attribute:

  1. <a> <a/>
  2. <area> <area/>
  3. <base> <base/>
  4. <link> <link/>

  1. The <a> <a/> tags are the most common way to use href
    <a> <a/> tags can have other tags (for example <p> <p/>) inside of them

    For example, ./exercises.html

  2. The <area><area/> tags are used to add a clickable link to an area of the screen. For example, part of this image will bring you to the exercises.html page

  3. The base tag is a way to set a default href value for all links on the page.
  4. The link tag is most often used for stylesheets and favicons. <link> is a self closing tag

Media Queries:

There are three types of media queries:

  • "@media screen"
  • "@media print"
  • "@media all"

"@media screen" works as an if statement

if "@ media screen and (min-width: 600px)" is true, apply {flex-direction: row}:

"If the viewport width > 600px, the apply flex-direction: row"

"@ media screen and (min-width: 600px)" and "@ media screen and (max-width: 800px)", apply "background-color: green":

"If 800px > viewport width > 600px, the apply flex-direction: row"

Gradients:

Three types of gradients:

  • Linear Gradients
  • Radial Gradients
  • Conic Gradients

Linear gradients are created through background: linear-gradient([color1], [color2]) on an element

This box has "background: linear-gradient(dodgerblue, yellow);"

To switch the color scheme order we can do "background: linear-gradient(to bottom, dodgerblue, yellow);"

Similarly, to make the color scheme go diagonal we can do "background: linear-gradient(to bottom right, dodgerblue, yellow);"
